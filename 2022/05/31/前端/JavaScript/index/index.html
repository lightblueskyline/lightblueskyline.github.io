事件机制

JavaScript 语言的一大特点就是单线程，同一时间只能做一件事情
指事件执行的规律
同步任务：在主线程上排队执行
异步任务：不进入主线程，而进入“任务队列”的任务，只有“任务队列”通知主线程，某个异步任务可以执行了，该任务才会进入主线程
